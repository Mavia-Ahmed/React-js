## 🚀 Getting Started

### 1️⃣ Clone the repository
```
git clone https://github.com/your-username/react-practice-repo.git
```

### 1️⃣ Navigate into the folder
```
cd react-practice-repo
```

### 3️⃣ Open a specific task or project
```
cd tasks/todo-app
```

### 4️⃣ Install dependencies
```
npm install
```

### 5️⃣ Run the project
```
npm start
```
